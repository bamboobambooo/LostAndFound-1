#失物招领
---
**1、新增失物招领**

	接口功能描述：新增事务招领
	
	URL/POST

	> 119.29.102.236/found/add
	
	入参（以下为必传参数，可传参数参考数据库）
	{
		"title": "liwanjun",
		"linkName": "123",
		"linkPhone": "13246874565",
		"goodType": "测试",
		"goodName": "测试",
		"descriptionDetail": "描述"
	}
	
	成功返回：
	{
		"result": "success",
		"data":1//该条记录的id
	}
	
	失败返回：
	{
	  "result": "fail",
	   "data":"错误原因"
	}
